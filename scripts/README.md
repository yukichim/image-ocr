# OCRモデル事前ダウンロード

初回実行時の待ち時間を解消するため、全てのOCRエンジンのモデルを事前にダウンロードできます。

## 使用方法

### 全エンジンのモデルをダウンロード

```bash
python scripts/download_models.py
```

### 特定のエンジンのみダウンロード

```bash
# PaddleOCRのみ
python scripts/download_models.py --engine paddleocr

# PaddleOCR-VL（レイアウト解析）のみ
python scripts/download_models.py --engine paddleocr_vl

# OnnxOCR（PP-OCRv5）のみ
python scripts/download_models.py --engine onnxocr
```

## ダウンロード先

モデルは以下のディレクトリに保存されます：

- PaddleOCR系: `~/.paddlex/official_models/`
- OnnxOCR: アプリケーション内蔵（ダウンロード不要）
- Tesseract: システムインストール済み

## 注意事項

- 初回ダウンロードには数分かかる場合があります
- ダウンロード済みのモデルは自動的にスキップされます
- インターネット接続が必要です
- PaddleOCR系のモデルは合計で約50MB程度のディスク容量を使用します
